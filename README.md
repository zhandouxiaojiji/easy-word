# 常用命名单词及缩写
+ [英语单词缩写规则](RULE.md)
+ [游戏开发](GAME.md)


|命名|缩写|翻译|说明|
|---|---|---|---|
|optimize|opt|优化||
|batch||批量处理||
|interval|intvl / inr|时间间隔||
|legacy||历史遗留|一般用在兼容旧代码提供的接口，提示这个接口会在不久的将来删除|
|template|tpl / tmpl|模板||
|segment|seg|段/分段||
|assembly|assy|集合||
|universal|univ|通用的||
|runtime||运行时|老外们是比较喜欢把跟所有与可执行程序相关的东西，包括库，执行环境等统统称为运行时|
